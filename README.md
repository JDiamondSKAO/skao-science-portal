## About

The Science Operations Portal serves as the primary reference for the international science community engaging with the Square Kilometre Array. It provides documentation on telescope access, observing procedures, data products, science verification activities, and the SRCNet distributed computing infrastructure.

The portal is built as a custom [Scroll Viewport](https://www.k15t.com/software/scroll-viewport) theme for Confluence Data Centre and is maintained by the SKAO Creative Production team.

## Key features

- Responsive layout with dark mode
- Full-text search with keyboard navigation
- Auto-generated table of contents with scroll-spy
- Dynamic announcements and deadline tracking
- WCAG 2.1 AA compliant
- Colour-coded telescope status indicators
