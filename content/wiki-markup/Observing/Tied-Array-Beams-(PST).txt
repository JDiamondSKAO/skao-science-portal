Tied-array beamforming is a signal processing technique that combines the signals from multiple antennas (or stations) by adding them together with a set of phase weights. The weights are chosen to ‘steer’ the array so that signals arriving from a given pointing direction on the sky add together coherently (i.e. add together in phase). The sensitivity of a tied-array beam increases linearly with the number of antennas in the array; however, the field of view decreases as the maximum distance between antennas in the array increases (proportional to ).

Tied-array beams form the basis of the SKA telescope’s pulsar timing (PST) and pulsar search (PSS) modes. While the implementation details differ, the PST and PSS systems on both telescopes use frequency-domain beamforming, in which digitized voltages from the antennas are split into frequency channels before being added together with a set of complex-valued phase weights.

Both the PST and PSS tied-array beamformers can combine signals from antennas within a 20-km maximum diameter. A set of subarray templates suitable for PSS/PST use are provided and can be selected.
