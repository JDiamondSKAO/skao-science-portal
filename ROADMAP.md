# Roadmap

Planned improvements for the SKAO Science Operations Portal. Subject to change based on SciOps team feedback and telescope commissioning timelines.

## Near-term

- Live telescope/system status feeds (API-driven, replacing static text)
- Announcement management via Confluence labels (no theme edits needed)
- Upcoming deadlines sourced from a shared calendar
- Enable Observing and Proposing sections when Full Operations phase begins
- Memo Series archive — 151 SKA Memos migrated to portal pages

## Medium-term

- Personalised views — recently visited pages and bookmarks for authenticated users
- Scoped search filters by content type (guides, tools, data docs)
- Quick links to recent observations or data releases
- Performance audit (Lighthouse) with documented target scores

## Accessibility

- Screen reader testing pass with NVDA and VoiceOver
- High contrast mode option beyond standard dark mode
- RTL layout support if required by future SKAO member states
